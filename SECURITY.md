# Security Policy

## Supported Versions
Only the latest released version is supported with security updates.

## Reporting a Vulnerability
If you discover a security issue, please report it privately.

- Preferred: GitHub Security Advisories (if enabled)
- Alternative: email security@<your-domain>

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested remediation

We will acknowledge receipt and provide a timeline for a fix.
